#### Sonoff Touch T1 1 Przycisk

| GPIO | Podłączenie |
| - | - |
| GPIO0	| TouchPad #1 |
| GPIO12 | Relay #1 oraz podświetlenie TouchPad'a #1 |
| GPIO13 | Niebieski LED |
| GPIO3	| RX pin (do użycia do zewnętrznych czujników) |
| GPIO1	| TX pin (do użycia do zewnętrznych czujników) |
